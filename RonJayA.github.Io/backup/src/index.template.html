Hello,

My name is Ron and I am a Web Developer as well as a Database Development Specialist. I develop website for companies using Oracle as the backend. I am also proficient in using server backends and developing front ends in Microsoft Access, PowerApps, and other languages. My SQL skills are highly developed and I code in Java, VBA, SAS, PowerApps, Power BI, C++ and Python. My Web Development proficiency is in front-end Web design in HTML5 and CSS/SCSS.

I work for Southern Califonia Edison as a Database Development Specialist and have developed applications for the Automation Database, NetComm Center, PCC App, TDRL Tool, Load Roll Tool, DVVC Tracker and Grid Mod Tool. Engineers are my basic users who address field issues in these applications as well as creating schemes for the projects mentioned. They also use them to in-service equipment and End Point Test it. My apps tend to pull in data sources from larger Edison databases so that my users have live data when they work that is always up to date. I do use Citrix for Microsoft Access applications as they run super quick on Citrix which is a Virtualized Desktop.

My other duties are Project Manager of NetComm Form Processing which adds new automated devices to the NetComm Database for radio controlled operation. I can even ping radios and controllers of equipment in code to make certain they are operable.

I have been with Southern California Edison for 4 1/2 years and really love my work. We also have a great team in Distribution Automation and I feel my skills are put to good use! Edison is great company to work for.
